# Bitops MindMap

## markmap cli USAGE

```shell
$ npx markmap-cli markmap.md
```

You can also install it globally:

```shell
$ yarn global add markmap-cli
$ markmap markmap.md
```

There is a watch mode so that you could edit the Markdown file and get updates on the fly:

```shell
$ markmap -w markmap.md
```
